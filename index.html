Photo-Real Rendering

The Fanta bottle was rendered in 3DS Max with realistic textures applied. Glass textures were applied to the main surface of the bottle.

Usage of Maya

The Dr Pepper can was created using a different 3D modelling program. Creating circles then lofting between them was used to create surfaces.

Usage of Blender

The Coffee Cup was created using a different 3D modelling program. The handle was created by selecting edge loops in the outward surface then bridging between them.

Assignment

The purpose of this project was to create a mobile first webpage with 3D modelling elements. This has been achieved through the usage of multiple 3D authoring packages, including 3DS Max, Maya, and Blender.

Persistence and MVC

The persistence layer has been achieved through using SQLite 3, hosted on the ITS webserver. Using a PHP interface, data can be fetched from the backend and rendered on the view / web layer. In this assignment this takes the form of the model descriptions and titles, which have been stored in the mobileapps.db and fetched when the user loads the model page.

Page Design
CSS, HTML, and Javascript has been used to make a responsive mobile first single page application. Heavy usage of the card bootstrap element allows for the containment of web page content and ensures that it resizes based on page size.
